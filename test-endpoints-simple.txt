API ENDPOINT TEST RESULTS
=========================

Run these commands manually to test:

1. Health Check:
curl http://127.0.0.1:8000/api/health

2. Register User:
curl -X POST http://127.0.0.1:8000/api/auth/register -H "Content-Type: application/json" -d "{\"name\":\"Admin User\",\"email\":\"admin@test.com\",\"password\":\"Admin@123\",\"role\":\"admin\"}"

3. Login:
curl -X POST http://127.0.0.1:8000/api/auth/login -H "Content-Type: application/json" -d "{\"email\":\"admin@test.com\",\"password\":\"Admin@123\"}"

4. Get Current User (use token from login):
curl http://127.0.0.1:8000/api/auth/me -H "Authorization: Bearer YOUR_TOKEN"

5. Get Users:
curl http://127.0.0.1:8000/api/users -H "Authorization: Bearer YOUR_TOKEN"

6. Get Settings:
curl http://127.0.0.1:8000/api/settings/consultation_fee -H "Authorization: Bearer YOUR_TOKEN"

7. Update Setting:
curl -X PUT http://127.0.0.1:8000/api/settings/consultation_fee -H "Authorization: Bearer YOUR_TOKEN" -H "Content-Type: application/json" -d "{\"value\":\"50000\"}"

8. Get Department Fees:
curl http://127.0.0.1:8000/api/departments/fees -H "Authorization: Bearer YOUR_TOKEN"

9. Get User Roles:
curl http://127.0.0.1:8000/api/users/roles?role=admin -H "Authorization: Bearer YOUR_TOKEN"

10. Bulk Import Services:
curl -X POST http://127.0.0.1:8000/api/services/bulk -H "Authorization: Bearer YOUR_TOKEN" -H "Content-Type: application/json" -d "{\"services\":[{\"service_name\":\"X-Ray\",\"service_type\":\"Imaging\",\"price\":50000}]}"
