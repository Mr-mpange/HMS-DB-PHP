========================================
DEPLOYMENT STEPS
========================================

1. Run deployment script:
   Windows: deploy.bat
   Linux/Mac: ./deploy.sh

2. Upload to Hostinger:
   - Frontend files ‚Üí public_html/
   - Backend files ‚Üí public_html/api/

3. Setup database:
   - Create database in Hostinger
   - Import backend/database_schema.sql

4. Configure backend:
   - Create public_html/api/.env
   - Set database credentials
   - Generate JWT_SECRET
   - Set production domain

5. Install & start:
   - SSH to server
   - cd public_html/api
   - npm install --production
   - pm2 start src/server.js --name hospital-api
   - pm2 save

6. Configure web server:
   - Create .htaccess for Apache
   - Enable SSL certificate

7. Test:
   - Visit https://yourdomain.com
   - Login with admin@hospital.com
   - Change default password!
